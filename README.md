# Make-up-website

A professional makeup artist website — **Glam by Heva**.

## Project Structure

```
Make-up-website/
├── index.html   # Main HTML page
├── style.css    # Styles
├── script.js    # JavaScript (navbar, gallery filter, contact form)
└── README.md
```

## Features

- **Hero section** with call-to-action buttons
- **About** section with stats
- **Services** listing with pricing
- **Gallery** with category filtering (All / Bridal / Editorial / Glam)
- **Testimonials** from past clients
- **Contact / Booking form** with basic validation
- Sticky navbar that changes style on scroll
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll-reveal animations

## Usage

Open `index.html` in any modern browser — no build step required.
